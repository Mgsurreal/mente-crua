# CMS v0.1 — Mente Crua

Este é o primeiro protótipo local do CMS.

## Como testar

1. Abra `cms/index.html` no navegador.
2. Escolha o módulo.
3. Preencha os campos.
4. Adicione blocos.
5. Clique em `Gerar`.
6. Baixe `index.html` e `data.json`.
7. Crie a pasta indicada pelo CMS.
8. Coloque os arquivos dentro dela.

Exemplo:

```txt
modules/artigos/a-ilusao-da-certeza/
├── index.html
├── data.json
└── img/
    ├── thumb.jpg
    └── imagem-01.jpg
```

## Módulos disponíveis neste protótipo

- Artigos
- Pensadores
- A Arte Explica
- A História Antes da Disney
- Personagens da Mitologia

## Importante

O navegador não consegue salvar arquivos automaticamente dentro da pasta do projeto sem permissão especial. Por isso, nesta versão, o CMS gera e baixa os arquivos. Depois colocamos manualmente na pasta certa.

Na próxima etapa, podemos criar um builder em Node.js para gerar tudo automaticamente.

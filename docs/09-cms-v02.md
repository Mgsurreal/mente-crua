# Mente Crua CMS v0.2

Primeira versão funcional do editor local.

## O que já faz

- Escolha de módulo: Artigo, Pensador, A Arte Explica, Antes da Disney e Mitologia.
- Campos principais: título, slug, subtítulo, categoria, thumb, autor e SEO.
- Editor por blocos:
  - parágrafo
  - título interno
  - imagem
  - citação
  - caixa destaque
  - lista
  - separador
- Reordenação de blocos com setas.
- Preview rápido na própria tela.
- Geração de `index.html`.
- Geração de `data.json`.
- Download separado dos dois arquivos.
- Sugestão automática do caminho da pasta final.

## Fluxo de uso

1. Abrir `cms/index.html`.
2. Escolher o módulo.
3. Preencher os campos.
4. Adicionar blocos.
5. Clicar em `Gerar / Atualizar preview`.
6. Baixar `index.html` e `data.json`.
7. Criar a pasta indicada, por exemplo:

```txt
modules/artigos/a-ilusao-da-certeza/
├── index.html
├── data.json
└── img/
```

8. Colocar as imagens dentro de `img/`.

## Limitação desta versão

Por segurança do navegador, o CMS ainda não grava direto dentro das pastas do projeto.
Nesta fase ele gera os arquivos para baixar.

A versão futura com Node.js poderá salvar automaticamente em:

```txt
modules/artigos/slug/index.html
modules/artigos/slug/data.json
```

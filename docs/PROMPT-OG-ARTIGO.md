# Prompt — imagem OG com texto

```text
Crie a imagem de compartilhamento OG de um artigo do portal Mente Crua usando como referência obrigatória a imagem de chamada sem texto já aprovada.

TEXTO EXATO
“[TÍTULO CURTO DO ARTIGO]”

FORMATO
- Dimensão exata: 1200 × 630 px.
- Uso: WhatsApp, Facebook, X e outras plataformas.
- Esta imagem não aparecerá visualmente no corpo do artigo.

COMPOSIÇÃO
- Preserve a mesma cena, personagem, identidade, iluminação, paleta e atmosfera da imagem de chamada.
- Adapte somente o enquadramento ao formato 1200 × 630.
- Posicione o título dentro da área segura central.
- Garanta leitura clara em miniatura.
- Mantenha elementos importantes afastados das bordas.

REGRAS
- Reproduza o título exatamente, sem alterar palavras.
- Não acrescentar subtítulo, chamada, endereço, logotipo, QR code ou marca-d’água.
- Não inventar uma cena diferente da imagem de chamada.
- Entregar somente a imagem plana, pronta para conversão em WebP.
```

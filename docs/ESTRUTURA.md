# Estrutura do clone Mente Crua CMS

Este pacote é um clone organizado. O original não foi alterado.

## Pastas principais

- `assets/css/` — CSS separado por função.
- `assets/img/` — imagens do site, artigos, social, hero, footer, pensadores e livros.
- `assets/js/` — JavaScript global do site.
- `content/` — base futura do CMS em JSON.
- `cms/` — protótipo local do formulário para criar artigos.
- `templates/` — reservado para templates do builder.
- `docs/` — documentação e anotações do projeto.
- `backup-original/` — cópia de segurança do CSS original.

## Como testar

Abra `index.html` com Live Server no VS Code.
Depois abra `cms/index.html` para testar o formulário de novo artigo.

## Observações

O HTML ainda continua estático nesta Sprint 1. A mudança principal foi organizar a base e deixar o caminho pronto para o CMS gerar artigos por JSON.

# Prompt — imagem larga de impacto

```text
Crie uma ilustração horizontal de impacto para uma passagem importante de um artigo do portal Mente Crua.

ARTIGO
Tema geral: “[TEMA]”
Passagem ilustrada: “[PASSAGEM OU VIRADA DO TEXTO]”
Sentimento ou ideia dominante: “[ATMOSFERA]”

FORMATO E USO
- Proporção horizontal aproximada: 16:7.
- Dimensão recomendada: 1600 × 700 px.
- A imagem ocupará toda a largura da coluna principal entre dois blocos de texto.
- Deve marcar uma mudança, expansão ou momento de impacto no artigo.

COMPOSIÇÃO
- Construa uma cena ampla com leitura clara de esquerda a direita.
- Preserve o ponto principal afastado das bordas.
- Utilize profundidade e escala sem criar ruído visual.
- Mantenha a mesma direção de arte, paleta e iluminação das demais imagens do artigo.

REGRAS
- Sem título, palavras, legenda desenhada, logotipo ou marca-d’água.
- Não transformar a imagem em outro banner.
- Não repetir a composição da thumb ou do banner.
- Entregar somente a imagem plana, pronta para WebP.
```
